# Gerador de Extratos — Motoristas Agregados

Ferramenta para geração de extratos de pagamento a partir do relatório exportado do TOTVS.

## Como usar

1. Acesse o link publicado pelo GitHub Pages
2. Importe o arquivo `.xls` exportado do TOTVS
3. Selecione os motoristas desejados
4. Gere e imprima os extratos em PDF

> Todos os dados ficam apenas no navegador do usuário. Nenhuma informação é enviada a servidores externos.
